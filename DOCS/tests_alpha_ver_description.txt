ТЕСТЫ:
Краткое описание альфа версии кода

Внешние данные и глобальные переменные

Внешние данные:
1. База данных SQLite (team_app.db):
   - Таблица users: Содержит информацию о пользователях, включая их никнейм и аватар.
   - Таблица topics: Содержит информацию о темах уроков, включая идентификатор и название темы.
   - Таблица tests: Содержит информацию о тестах, включая идентификатор, название теста и тип теста.
   - Таблица words: Содержит информацию о словах, включая их английский и русский варианты, изображения и аудио.
   - Таблица words_groupings: Связывает слова с темами.

Глобальные переменные:
- ACTIVE_USER (dict): Словарь, содержащий информацию о текущем пользователе.
  - nickname: Строка, представляющая никнейм пользователя.
  - image: Двоичные данные изображения пользователя (если есть) или None.

Определенные классы и их методы

1. TopicSelectionScreen:
   - __init__: Инициализирует экран выбора темы урока.
   - setup_ui: Настраивает пользовательский интерфейс.
   - show_user_info: Отображает информацию о текущем пользователе.
   - load_topics: Загружает список тем из базы данных.
   - load_tests: Загружает список тестов из базы данных.
   - clear_window: Очищает окно от всех виджетов.
   - select_topic: Обрабатывает выбор темы.
   - select_test: Обрабатывает выбор теста.
   - start_test: Запускает выбранный тест.

2. BaseLessonScreen:
   - __init__: Инициализирует базовый экран урока.
   - setup_ui: Настраивает пользовательский интерфейс.
   - show_user_info: Отображает информацию о текущем пользователе.
   - create_progress_bar: Создает прогресс-бар из 8 элементов.
   - update_progress: Обновляет состояние прогресс-бара.
   - clear_window: Очищает окно от всех виджетов.
   - return_to_selection: Возвращает пользователя к экрану выбора темы.
   - prev_word: Обрабатывает переход к предыдущему слову (неактивно).
   - next_word: Обрабатывает переход к следующему слову.
   - check_answer: Проверяет выбранный или введенный ответ.
   - show_word: Отображает текущее слово.
   - play_sound: Воспроизводит аудиофайл.
   - check_music_end: Проверяет окончание воспроизведения аудио.
   - cleanup_temp_file: Удаляет временный аудиофайл.

3. WordLessonScreen (наследуется от BaseLessonScreen):
   - __init__: Инициализирует экран урока.
   - load_words: Загружает слова для урока из базы данных.
   - show_word: Отображает текущее слово и варианты перевода.

4. WordWriteScreen (наследуется от BaseLessonScreen):
   - __init__: Инициализирует экран урока.
   - load_words: Загружает слова для урока из базы данных.
   - show_word: Отображает текущее слово и текстовое поле для ввода перевода.

5. WordAudioScreen (наследуется от BaseLessonScreen):
   - __init__: Инициализирует экран урока.
   - load_words: Загружает слова для урока из базы данных.
   - show_word: Отображает серый квадрат и кнопки для выбора перевода по аудио.

6. MainTestApp:
   - __init__: Инициализирует основное приложение.
   - show_topic_selection_screen: Показывает экран выбора темы урока.
   - start_lesson: Запускает соответствующий экран урока на основе выбранного типа теста.

Окна и переходы между ними

1. Экран выбора темы урока (TopicSelectionScreen):
   - Пользователь может выбрать тему урока и тест.
   - Доступные кнопки: "Начать тестирование", "Выход", "Назад" (неактивна).
   - Переход к окну урока после выбора темы и теста и нажатия на "Начать тестирование".

2. Экран урока (WordLessonScreen, WordWriteScreen, WordAudioScreen):
   - Отображает текущее слово и варианты перевода или текстовое поле для ввода перевода.
   - Доступные кнопки: "Предыдущее слово" (неактивна), "Следующее слово", "Выход", "Выбрать тест".
   - Переход обратно к экрану выбора темы после нажатия на "Выбрать тест".

Логика тестов и их реализация

1. Тест на выбор правильного перевода (WordLessonScreen):
   - Пользователю показывается слово на русском языке и 6 вариантов перевода на английский, один из которых правильный.
   - После выбора варианта отображается результат ("Правильно!" или "Неправильно!").
   - Варианты перевода выбираются случайным образом из всех слов данной категории для каждого нового слова.

2. Тест на написание слова на английском (WordWriteScreen):
   - Пользователю показывается слово на русском языке и текстовое поле для ввода перевода на английский.
   - После ввода перевода и нажатия кнопки "Проверить" отображается результат ("Правильно!" или "Неправильно!").

3. Тест на выбор правильного перевода по аудио (WordAudioScreen):
   - Пользователю воспроизводится аудио с произношением слова на английском языке и показывается серый квадрат.
   - Пользователь выбирает правильный перевод на русском языке из 6 вариантов.
   - После выбора варианта отображается результат ("Правильно!" или "Неправильно!").
   - Варианты перевода выбираются случайным образом из всех слов данной категории для каждого нового слова.

Элементы пользовательского интерфейса и их функции

1. TopicSelectionScreen:
   - Кнопки выбора темы: Позволяют выбрать тему урока.
   - Кнопки выбора теста: Позволяют выбрать тип теста.
   - Кнопка "Начать тестирование": Запускает выбранный тест.
   - Кнопка "Выход": Закрывает приложение.
   - Кнопка "Назад": Неактивна, функционал будет добавлен позднее.
   - Пользовательская информация: Отображает аватар и никнейм текущего пользователя.

2. BaseLessonScreen и производные классы:
   - Пользовательская информация: Отображает аватар и никнейм текущего пользователя.
   - Прогресс-бар: Показывает прогресс выполнения теста.
   - Сообщения ("Правильно!"/"Неправильно!"): Отображаются после выбора или ввода ответа.
   - Кнопка "Предыдущее слово": Неактивна.
   - Кнопка "Следующее слово": Переход к следующему слову.
   - Кнопка "Выход": Закрывает приложение.
   - Кнопка "Выбрать тест": Возвращает к экрану выбора темы.

Этот краткий обзор "альфа версии" описывает основные компоненты, логику и пользовательский интерфейс программы.
